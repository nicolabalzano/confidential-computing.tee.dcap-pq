# PCE source code
The PCE is part of [Intel(R) Software Guard Extensions for Linux\* OS](https://github.com/intel/confidential-computing.sgx/) Github repository. The libsgx_pce.signed.so in prebuilt package is built by [pce](https://github.com/intel/confidential-computing.sgx/tree/main/psw/ae/pce) with branch [sgx_2.25_reproducible](https://github.com/intel/confidential-computing.sgx/tree/sgx_2.25_reproducible) and signed by Intel.
